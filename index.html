Goals: Operating System

Speed: Never forget that a 4-core 1.5GHz processor is incredibly powerful.

Simplicity: You can comprehend the entire system in one night.

Interactivity: The system should know what you want to do before you do.

Self-Contained: All components are made in-house, in one cohesive language.

Modularity: Remove any piece of code or program you don't want.

Documented: If you don't understand something, the system will explain it.

S-expressions: Lisp-style parentheses make for a simple compiler.

Grounded: Programming the underlying machine is paramount.

Full-stack: From bit-banging to the user interface shell, all-in-one.

Minimalistic: Manual memory management is the only kind provided.

Improvable: Version two will allow for implicit parentheses per line and indentation-based nested expressions
